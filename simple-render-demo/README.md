# Simple Render Demo 🚀

**Backend**: Node/Express API (no DB)
**Frontend**: React CRA static site

## Local Test
```
cd backend && npm i && npm start  # port 3001
cd ../frontend && npm i && npm start  # port 3000
```

## Render Deploy (Free)
1. Backend Web Service (dir: backend) - npm install | npm start
2. Frontend Static Site (dir: frontend) - npm install && npm run build | build/
3. Frontend env: REACT_APP_API_URL = your-backend.onrender.com

Live demo shows API working!
